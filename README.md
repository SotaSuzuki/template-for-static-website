# template-for-static-websites

This is made for making a static websites which is pretty tiny, and is as a play ground for trying out new features.

## Features

- Dev Environment
  - Gulp
  - BrowserSync

- HTML
  - htmlhint

- CSS
  - PostCSS
  - stylelint

- JS
  - nothing!

### File Structure

```

Project
│
├── gulp_tasks
│     └ [ task files ]
│
│── src
│     ├ assets
│     └ index.html
│
├── .csscomb.json
├── .editorconfig
├── .gitignore
├── stylelintrc
├── gulpgile.js
├── package.json
└── README.md

```


## Usage

### Set up

install npm modules:

```bash
npm i
```

### Run

```bash
npm start
```

then access to [http://localhost:3000][1]
